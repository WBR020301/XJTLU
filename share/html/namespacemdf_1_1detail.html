Support function for opening an MDF file.

Support function that opens an MDF file. This may fail if both a writer and a reader tries to use the file at the same time.

Parameters
fileReference to a file stream pointer.
filenameFull path to file.
modeOpen mode.
Returns
True if the file was opened.
